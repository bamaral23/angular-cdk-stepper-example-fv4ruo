Angular CDK Stepper Example

The CDK Stepper aims to provide basic functionality for a stepper / form wizard, such as selecting and displaying the currently active step.

Where are the docs?

There IS an example on the material page, it's just in a bad place. Here it is: https://material.angular.io/guide/creating-a-custom-stepper-using-the-cdk-stepper ...

How to get it running?

  1. Install @angular/cdk and add CdkStepperModule to your module.
  2. Create a component my-comp that extends CdkStepper. In this component, you should:
    • Provide a footer / header for the general overview over the steps
    • Provide the possibilty to navigate in the header / footer (next / back buttons)
    • Add [ngTemplateOutlet]="selected.content" somewhere. The cdkStep will be projected there...
  3. Add code app-my-comp with cdk-step's inside

What features does the stepper have?

  • Displaying only the active step. When this.selectedIndex is changed, the the new step is displayed.
  • Exposing some properties of the steps:
    • selectedIndex (can be set and get)
    • steps (info about the steps)
  • Working stepper with just some html for the actual step, no typescript required.
  • A "linear mode", where you can only progress if you have completed previous steps. Also integrates well with forms (I think).
